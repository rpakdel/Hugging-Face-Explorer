## Packages
@xenova/transformers | Run machine learning models in the browser
framer-motion | Smooth animations for result reveals and transitions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
The application runs ML models client-side using WebAssembly via transformers.js.
History is saved to the backend via POST /api/operations.
